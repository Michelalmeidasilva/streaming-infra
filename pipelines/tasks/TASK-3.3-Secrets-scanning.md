# Tarefa 3.3: Secrets scanning

## 📌 Épico
**Segurança (SAST, SCA, Secrets)**

## 🎯 Objetivo
Detectar credenciais e tokens expostos no código.

## 🧠 Contexto e Embasamento (RESEARCH)
Garantir que senhas e chaves não fiquem hardcoded. Ferramentas como Gitleaks bloqueiam commits contendo essas vulnerabilidades.

## 📋 Requisitos
- Integrar Gitleaks
- Bloquear commit com secrets
- Criar política de rotação de credenciais
