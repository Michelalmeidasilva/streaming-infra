# Tarefa 4.1: Detecção de duplicação

## 📌 Épico
**Duplicação de Código**

## 🎯 Objetivo
Identificar código duplicado que aumenta a complexidade e bugs.

## 🧠 Contexto e Embasamento (RESEARCH)
Código duplicado leva a correções incompletas e aumenta o custo técnico. SonarQube medirá e alertará sobre duplicação em PRs (limite sugerido < 3%).

## 📋 Requisitos
- Habilitar duplicação no SonarQube
- Definir threshold (ex: 3%)
- Exibir no PR
