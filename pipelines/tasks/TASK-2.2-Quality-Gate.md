# Tarefa 2.2: Quality Gate

## 📌 Épico
**Code Quality & Code Smells**

## 🎯 Objetivo
Bloquear código de baixa qualidade de entrar na branch principal.

## 🧠 Contexto e Embasamento (RESEARCH)
Guardrails tradicionais envolvem quality gates que bloqueiam o merge se os limites (ex: cobertura ou smells) forem excedidos.

## 📋 Requisitos
- Definir regras: Coverage mínimo (ex: 80%), Code smells críticos = 0
- Configurar bloqueio de PR
- Testar cenário de falha
